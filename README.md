# Drone-Marketing-Survey-Analytics-AHP-Competency-Weighting
This project provides a runnable reference implementation that reproduces the core analytics described in the paper. The script loads the paper’s raw survey dataset from a CSV file and prints a complete, non-truncated set of descriptive statistics that align with the Results section, including the binary outcome summaries used in Figure 6, the multi-select shopping channel adoption statistics used in Figure 7, and the marketing evaluation plus multi-select improvement items used in Table 2. The printing layer explicitly disables common pandas truncation settings so that all rows and columns are shown in full in the terminal output.

The same script also implements the paper’s competency-weighting algorithm using the Analytic Hierarchy Process (AHP). It takes the pairwise comparison matrix provided in Figure 8, computes the principal eigenvector to obtain normalized primary indicator weights, and reports AHP consistency diagnostics including the principal eigenvalue (lambda_max), the Consistency Index (CI), and the Consistency Ratio (CR) using the standard Saaty Random Index table. After that, the algorithm combines the computed primary weights with the secondary indicator weights given in Table 3, producing global weights for all secondary indicators and normalizing them to sum to one for reporting. This end-to-end flow enables direct replication of the competency weighting output from the matrices and weights described in the manuscript.

To run the script, Python is required along with pandas and numpy. You can install dependencies with pip install pandas numpy. The script accepts an optional command-line argument for the dataset path. If the CSV is in the same folder as the script, running python paper_algorithm_run.py will load it by default. If the CSV is elsewhere, running python paper_algorithm_run.py /path/to/drone_marketing_raw_synthetic_dataset.csv will load the specified file. The output is printed to stdout as full tables and includes both descriptive statistics and AHP weighting results, so it can be copied directly into an appendix or used for verification during peer review.
